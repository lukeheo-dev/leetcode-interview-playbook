# LeetCode Interview Playbook Roadmap (Pattern-based)

## Rules
- Pattern-first. Each problem must be placed under a Phase + Pattern folder.
- For every problem, write: (1) approach, (2) complexity, (3) pitfalls, (4) short English summary.
- Every solved problem must have a `notes.md`.
- End of each phase: timed mixed set + review.

## Phases
- Phase 00: Reset
- Phase 01: Arrays & Hashing
- Phase 02: Two Pointers & Sliding Window
- Phase 03: Stack / Monotonic Stack / Intervals
- Phase 04: Linked List / Binary Search (Advanced)
- Phase 05: Trees / BFS / DFS
- Phase 06: Heap / Greedy basics
- Phase 07: Backtracking
- Phase 08: Graphs
- Phase 09: DP
- Phase 10: Bit / Math / Mixed Review
